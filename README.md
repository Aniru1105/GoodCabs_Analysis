## üìå Project Overview
Goodcabs, a cab service company, has rapidly established itself in India's tier-2 cities, providing sustainable income opportunities for local drivers while ensuring high-quality service to passengers. As the company sets ambitious growth and performance targets for 2024, this project aims to analyze key operational metrics to guide strategic decision-making.

## üéØ Project Objectives
- **Trip Volume Analysis** ‚Äì Evaluating ride demand and city-wise distribution.
- **Passenger Satisfaction** ‚Äì Assessing ratings, feedback, and customer sentiment.
- **Repeat Passenger Rate** ‚Äì Identifying loyalty trends and retention challenges.
- **Trip Distribution Insights** ‚Äì Understanding peak hours, location hotspots, and ride types.
- **New vs. Repeat Passenger Ratio** ‚Äì Balancing user acquisition with retention.
